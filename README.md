# 🚀 Task Manager App

A full-stack MERN application to manage daily tasks efficiently.

## 🔥 Features

* 🔐 User Authentication (JWT)
* ✅ Create, Update, Delete Tasks
* 🌙 Dark Mode UI
* ⚡ Fast and Responsive UI

## 🛠 Tech Stack

* Frontend: React.js
* Backend: Node.js, Express.js
* Database: MongoDB Atlas

## 📂 Project Structure

* frontend → React App
* backend → API & Database
